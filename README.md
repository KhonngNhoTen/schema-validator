# Schema Validator

*This is a library validates object in javascript*

Use schemas to wrap object properties and provide functions to modify the schema as needed.

## Install
Install with **NPM:**

```js
npm install @cua/schema-validator
```

## Features
- Schema
- Validator
  - validate
  - validateAsyn
- SchemaOptionHelper
